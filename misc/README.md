# Misc

This folder groups non-runtime project materials that do not need to live at the repository root.

## Folders

- `play-store/`
  Google Play Console drafts and submission support text.
- `legal/`
  Privacy policy drafts and legal copy.
- `backend/`
  Loose backend reference notes and endpoint summaries.
- `ai/`
  AI-generated planning or refactor briefs kept for reference.
- `references/`
  Supporting files such as PDFs and other external reference material.
